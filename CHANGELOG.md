## [1.0.0] - 2025-01-18
### Added
- Initial stable release
- POML-driven README structuring engine
- Support for TXT, JSON, PDF, DOCX, RTF, HTML, XML inputs
- Deterministic, order-independent section detection
- JSON flattening without artifacts or inference
- Clean Markdown README output
- Streamlit-based web UI
- Modular project architecture
- Pytest test suite
- Docker support
- GitHub Actions CI pipeline
- Runtime footer with version, commit hash, and build timestamp
- POML rules as the single source of truth
